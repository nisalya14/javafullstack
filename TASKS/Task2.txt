1. How to comment on HTML tags?

start with <! -- and end with -->

Example:
<!DOCTYPE html>
<html>
<body>

<!-- This is a comment -->
<p>Comment in HTML</p>


</body>
</html>


2. How to include abbreviations?

The tag <abbr> was used to include the abbreviation in the Html.



<!DOCTYPE html>
<html>
<body>
<h1>Abbreviations</h1>
<p>I'm complete my <abbr title="Bachlor of Engineering">BE</abbr>in 2024</p>
</body>
</html>
